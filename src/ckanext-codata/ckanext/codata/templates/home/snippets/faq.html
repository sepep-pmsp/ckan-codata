Perguntas Frequentes

Quais dados são disponibilizados no URBIS?
keyboard_arrow_up

O Portal URBIS centraliza uma vasta gama de dados públicos focados no desenvolvimento e gestão urbana. O nosso repositório inclui informações detalhadas sobre:

  • Planeamento Urbano: Zoneamento, uso do solo e infraestruturas.
  • Mobilidade: Transportes públicos, tráfego e logística.
  • Meio Ambiente: Qualidade do ar, áreas verdes e saneamento.

Os dados são disponibilizados em formatos abertos e legíveis por máquina (como CSV, JSON, GeoJSON), garantindo que podem ser facilmente reutilizados para análises, estudos académicos ou desenvolvimento de aplicações.

Como funciona o processo de atualização dos dados?
keyboard_arrow_up

Para garantir a integridade e a frescura da informação, utilizamos uma arquitetura de medalhão. Os dados passam por camadas de validação e transformação — desde a ingestão bruta até ao tratamento final.

Todo este fluxo de coleta, transformação e publicação é orquestrado pelo Dagster, o que nos permite manter o catálogo sempre atualizado de forma eficiente.

Quais são as formas de consulta dos dados?
keyboard_arrow_up

O URBIS foi desenhado para atender a diferentes perfis de utilizadores. Existem duas formas de acesso aos dados:

  • Interface Web: Navegando visualmente pelo portal, usar a barra de pesquisa, filtrar por etiquetas e pré-visualizar tabelas ou mapas diretamente no navegador antes de fazer o download.
  • API REST: Para programadores e cientistas de dados, oferecemos acesso direto e automatizado a todos os datasets através da API do CKAN, permitindo a integração dos nossos dados em sistemas externos em tempo real.
Como utilizar a API do portal URBIS?
keyboard_arrow_up

Com ela, é possível:

  • Consultar todos os metadados do portal;
  • Baixar todos os arquivos do portal.
